# Knihovna ikon pro prezentace

Kurátorovaná kolekce 200+ inline SVG ikon z [Lucide](https://lucide.dev/) (MIT licence). Všechny ikony jsou 24x24, stroke-based, používají `currentColor` — automaticky přebírají barvu rodiče.

## Jak používat

Vlož SVG přímo do HTML. Velikost ovládej přes `width`/`height`, barvu přes CSS `color` na rodiči:

```html
<div style="color: var(--accent);">
  <!-- ikona automaticky přebere accent barvu -->
  <svg xmlns="http://www.w3.org/2000/svg" width="32" height="32" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round">...</svg>
</div>
```

Pro responzivní velikost: `width="clamp(24px, 3vw, 48px)"` nefunguje v atributu — použij CSS:
```css
.icon { width: clamp(24px, 3vw, 48px); height: clamp(24px, 3vw, 48px); }
```

### Tipy pro styling

- **Barva**: Nastav `color` na rodičovském elementu — ikona ji zdědí přes `currentColor`
- **Velikost**: Změň `width` a `height` atributy nebo CSS. `viewBox` nech vždy `0 0 24 24`
- **Tloušťka čáry**: Změň `stroke-width` (výchozí je `2`, zkus `1.5` pro tenčí nebo `2.5` pro výraznější)
- **Animace**: Přidej CSS transitions/animations — fungují na SVG elementech normálně
- **Přístupnost**: Přidej `aria-hidden="true"` pokud je ikona dekorativní, nebo `role="img" aria-label="popis"` pokud nese význam

```css
/* Příklad: ikona s hover efektem */
.icon-interactive {
  transition: transform 0.2s ease, color 0.2s ease;
  cursor: pointer;
}
.icon-interactive:hover {
  transform: scale(1.1);
  color: var(--accent);
}
```
