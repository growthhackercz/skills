# Video Production

## Purpose
Turn an approved marketing message into a production-ready video plan and assets package for short-form or campaign video.

## Use when
- a message needs video adaptation
- campaign execution requires short-form creative
- content production needs structure before editing

## Inputs
- script or core message
- brand rules
- target channel and format
- proof assets or B-roll availability
- CTA objective

## Outputs
Return:
- video concept
- scene/shot structure
- asset list
- on-screen text suggestions
- edit notes
- delivery variants by channel

## Rules
- optimize for message clarity first
- respect platform format
- keep production scope realistic

## Integrations
Ideal integrations:
- asset library
- script repository
- publishing calendar
- video editing workflow
