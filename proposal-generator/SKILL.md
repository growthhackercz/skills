# Proposal Generator

## Purpose
Create structured, accurate proposal drafts tailored to opportunity context while staying inside approved offer boundaries.

## Use when
- a qualified opportunity is ready for a proposal
- the team needs a consistent proposal structure

## Inputs
- lead summary
- problem / desired outcome
- approved offer components
- pricing / packaging rules
- scope constraints
- implementation notes

## Outputs
Return:
- proposal structure
- personalized summary
- solution framing
- scope and assumptions
- next-step instructions
- open variables needing approval

## Rules
- no invented pricing or commitments
- surface assumptions explicitly
- keep the document skimmable and decision-friendly

## Integrations
Ideal integrations:
- CRM opportunity
- pricing system
- template repository
- delivery / onboarding constraints
